## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

- **Content**: Edit `src/data/content.ts` to update the destination data and marketing text.
- **Images**: Place your `.webp` images in the `public/images/` folder and update the paths in `src/data/content.ts`.
